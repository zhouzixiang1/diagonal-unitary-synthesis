## 一、研究背景与意义

量子线路综合问题是连接量子算法设计与量子硬件实现的桥梁，在当前含噪声中等规模量子（NISQ）时代尤为关键。对角酉矩阵作为一类基础且泛用的酉变换，广泛出现在量子态制备、Hamiltonian模拟（可对易Pauli-Z项）、QAOA的相位分离算子、IQP线路等场景中，其综合效率直接影响上层量子算法的可执行性。

已有研究表明，在NISQ设备上适度近似原始算法（以可控误差换取更少的量子资源消耗），实际线路的保真度反而可能超过精确算法的实现。这为"近似综合"提供了坚实的实践动机，也是两份基础工作共同的核心命题。

与此同时，量子计算对经典密码体系的潜在威胁（Shor算法之于RSA/ECC，Grover算法之于对称密码）已使量子密码分析资源估计成为网络空间安全领域的热点方向。近两年出现了大量将变分量子算法（尤其是QAOA）直接用于密码分析（已知明文攻击建模为MAX-SAT/QUBO问题）的工作，这类线路的核心组件——相位分离算子——本质上就是一个对角酉矩阵。这为"近似对角酉矩阵综合"这一相对纯粹的量子计算理论工作，提供了一条通向网络空间安全专业实际问题的具体路径。

## 二、国内外研究现状

### 2.1 对角酉矩阵综合算法的演进

近两年的研究重心已从早期的"CNOT门计数优化"逐步转向"电路深度优化"与"硬件连通性约束"：

- 深度最优的精确综合算法，通过统一的线路重写规则将深度较此前方法（Welch's method）近乎减半 [5]
- 面向qudit系统的相位构件方法，10-qutrit对角酉矩阵可将深度从约10万压缩到500（以300个辅助qutrit为代价），兼顾NISQ与容错时代 [6]
- 面向二维网格连通性约束的对角酉矩阵综合（2026年最新工作），明确指出对角酉矩阵是QAOA相位分离算子、可对易Pauli-Z哈密顿量模拟、IQP线路的共同基础构件 [7]
- 基于生成流网络（GFlowNet）的多样化酉综合方法 QFlowNet，探索"一题多解"的综合范式 [8]
- 利用反射对称性专门优化特定结构对角酉矩阵的综合算法 [9]

**观察**：上述工作几乎全部聚焦精确综合，"近似换资源"这一维度在近期文献中鲜有涉及——这正是两份基础工作最独特、目前仍是空白的贡献点。

### 2.2 蒙特卡洛树搜索在量子线路综合/编译中的应用

- **MonteQ**（2026）：面向Hamiltonian模拟线路综合，采用"上层MCTS搜索Pauli旋转排序 + 下层综合启发式"的两层设计，将排序问题的阶乘级搜索空间转化为可控的树搜索，相比Rustiq等编译器平均节省30%的CNOT门 [3]。该工作是目前与本课题最接近的"竞品"，其MCTS面向的是精确综合（仅可选择性放松酉保持约束），未显式建模误差-资源权衡曲线，也未讨论线路深度。
- **PWMCTS**（2025）：面向参数化量子线路结构设计，提出渐进加宽（progressive widening）技术动态扩展搜索树的动作空间，缓解传统MCTS分支因子过大的问题 [4]。
- 早期工作已将MCTS用于qubit映射/线路变换问题，验证了树搜索范式在量子编译问题上的有效性 [17]（已被两份源文档引用）。

### 2.3 强化学习等竞争范式

2025-2026年间，强化学习已成为量子线路综合领域比MCTS更活跃的研究范式：QLSynth 在QFT/Grover/Shor等基准上实现30%-50%门数削减、40%深度削减 [10]；面向Clifford线路的等变强化学习可扩展到30比特规模 [11]；面向优化pass组合的自适应强化学习 [12]；面向Pauli网络的强化学习综合，两比特门数削减达2倍且单次执行低于10毫秒 [13]。

**启示**：RL需要训练、可解释性较弱；MCTS无需训练、天然支持"给定资源找最优解"的anytime搜索，与两份基础工作"输入CNOT数/输入误差"的任务设定高度契合——这是本课题坚持MCTS路线而非转向RL的核心理由，也是文献综述与答辩中需要正面回应的问题。

### 2.4 量子计算与密码分析的交叉

- 使用QAOA攻击Heys玩具密码，将已知明文密钥恢复问题转化为MAX-SAT实例 [14]
- 面向Trivium密码（80比特密钥）的QAOA云端优化攻击隐私风险研究 [15]
- 面向S-AES的增强变分量子攻击算法（Enhanced VQAA），发表于ACM Transactions on Quantum Computing (2025) [16]
- 大量2025-2026年工作聚焦Grover密钥搜索的资源估计（ZUC、AES、S盒电路等），是当前密码学与量子计算交叉最活跃的子方向

**技术说明（重要）**：Grover标记预言机 U_f|x⟩=(-1)^{f(x)}|x⟩ 本质是对角元素限定在{+1,-1}的对角酉矩阵特例，其所有相位"同等重要"，不存在"近似省略次要相位"的空间，与本课题近似综合方法的核心机制不匹配。相比之下，QAOA相位分离算子 U_C=diag(e^{-iγC(x)}) 的相位由代价函数C(x)连续决定，不同x的相位重要性天然分层，与"挑选高重要性相位、近似省略低重要性相位"的方法论高度吻合，因此本课题选择QAOA密码分析场景而非Grover预言机场景作为应用验证对象。

## 三、现有工作分析

### 3.1 毕业论文方法拆解

核心是蒙特卡洛树搜索：每个节点对应α矩阵中的一个候选相位位置，属性包括 value、child、ancestor、father、visit、quality。节点选择公式：

```
w_i = β1·(quality_i/visit_i) + β2·Σ(n-d(i→j))·α_j·[visit_j=0] + β3·√(visitFather_i/visit_i)
```

三项分别对应"现有解质量"“未来潜力（汉明距离加权）”“探索项”，β1、β2、β3 为人工设定的固定权重。参数（α）优化使用梯度下降，且延后到树搜索完成后统一进行以节省开销。最高可综合12比特。

### 3.2 arXiv论文方法拆解

将排序问题转化为无向图上的路径搜索：先计算"相位重要性"（|α|标准化后经logistic函数锐化关键阈值附近的差异），再依次执行路径选择（贪心选择未访问的高重要性邻居）、路径扩展（借鉴Hamiltonian路径判定思想处理受阻情形）、"死胡同"处理（综合自身重要性与到其他活跃节点的距离评分）。最高可综合15比特，并提出"算法效用比"（CNOT节省比例/误差）作为量化评价指标。

### 3.3 两者对比

| 维度 | 毕业论文（MCTS） | arXiv论文（图路径搜索） |
|---|---|---|
| 核心方法 | 蒙特卡洛树搜索 | 相位重要性 + 确定性图路径搜索 |
| 节点/路径评价 | 人工加权公式（β1β2β3） | 闭式相位重要性得分（标准化+logistic） |
| 最大比特数 | 12 | 15 |
| 是否量化"效用比" | 否 | 是 |
| 是否讨论线路深度 | 否 | 否 |
| 参数优化时机 | 搜索完成后统一梯度下降 | 同样延后优化 |

### 3.4 可继承与可改进之处

1. 毕业论文的MCTS选择公式完全依赖人工调参，可用arXiv论文的相位重要性得分作为原则性先验替代/补充，理论上应能提升搜索效率与可扩展性；
2. 两份工作均未讨论渐进加宽等现代MCTS增强技术，也未触及线路深度这一近两年的研究热点维度；
3. 两份工作均未与具体应用场景结合，停留在随机对角酉矩阵的基准测试。

## 四、研究目标与创新点

**研究目标**：设计一种融合相位重要性先验与渐进加宽机制的改进MCTS算法，用于近似对角酉矩阵综合，在CNOT门数、算法误差、可扩展比特数等指标上超越原始MCTS方法，并将其应用于QAOA密码分析攻击线路的资源优化，给出资源-保真度-攻击成功率的量化权衡分析。

**创新点**：
1. 提出相位重要性引导的MCTS节点选择策略，以闭式相位重要性得分作为搜索先验，替代人工加权公式，兼具MCTS的通用搜索能力与确定性方法的高效指导（融合两份基础工作的核心优势，而非简单复现其一）；
2. 引入渐进加宽机制缓解搜索树分支爆炸，冲击更大规模比特数（目标超越原始工作12比特的上限）；
3. 首次将近似对角酉矩阵综合方法应用于QAOA密码分析攻击场景，为量子密码分析的资源估计提供新的方法学工具。

## 五、技术方案

### 5.1 相位重要性引导的MCTS节点选择

将相位重要性得分 Imp[i]（标准化|α_i|后经logistic锐化）引入节点选择公式，替代原公式中依赖人工调参的质量项，作为搜索的先验概率（类比AlphaZero中策略网络提供的先验），使搜索更快收敛到高价值分支；同时保留visit/quality的自适应更新机制以修正先验偏差。

### 5.2 渐进加宽

借鉴PWMCTS的思路 [4]，控制节点实际展开的子节点数量随访问次数呈次线性增长（如 k = C·N(parent)^α），避免比特数增大时子节点数量线性增长导致的分支爆炸，是冲击15比特以上规模的关键手段。

### 5.3 参数优化环节

沿用"搜索阶段用λ的二范数作代理评价、搜索完成后再用e^{iλ}的二范数做梯度下降精修"的两阶段策略，视中期实验结果决定是否需要进一步优化（如引入更快的二阶优化方法）。

### 5.4 应用场景设计：QAOA密码分析攻击线路优化

复现一个玩具密码（参照Heys密码或自行设计的小规模SPN/Feistel结构）的已知明文密钥恢复攻击，将其建模为MAX-SAT/QUBO实例并构造对应QAOA相位分离算子，作为待综合的目标对角酉矩阵。应用本课题方法对该相位分离算子做近似综合，报告CNOT门数/线路深度的节省比例与攻击成功概率损失之间的权衡曲线。比特规模控制在10-20量级，与方法本身的可扩展性区间保持一致，明确将其定位为方法论验证的概念验证（proof-of-concept），不涉及对真实规模密码（如AES-128）的实际攻击。

## 六、实验设计

### 6.1 对比基线（均可依据两份源文档已给出的完整伪代码复现，无需外部代码）

| 方法 | 类型 | 来源 |
|---|---|---|
| 精确综合（闭式CNOT分解） | 精确 | 经典结果 |
| 原始MCTS | 近似 | 毕业论文 Algorithm 1 |
| 相位重要性图路径搜索 | 近似 | arXiv论文 Algorithm 1-6 |
| 本文方法（相位重要性MCTS + 渐进加宽） | 近似 | 本课题 |

MonteQ、QLSynth等RL/MCTS方法因问题设定不同（面向通用Pauli网络/精确综合），在相关工作中定性讨论，不作为强制复现的实验基线。

### 6.2 测试场景

- **场景一（核心方法验证）**：随机生成的对角酉矩阵，比特数覆盖8/10/12/15（对齐两份源文档协议，可直接与已发表数值比较），CNOT削减比例5%-50%。
- **场景二（应用验证）**：玩具密码QAOA攻击线路的相位分离算子，比特数10-20。

### 6.3 评价指标

CNOT门数、线路深度、综合误差 D(U_C,U_T)、算法效用比（CNOT节省比例/误差，沿用arXiv论文定义）、运行时间；场景二额外报告攻击成功概率相对损失。

### 6.4 消融实验

分别验证"仅加入相位重要性先验（不加渐进加宽）”“仅加入渐进加宽（不加相位重要性先验）”“两者都加入”三种配置，量化两个改进点各自的贡献。
