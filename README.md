# Diagonal Unitary Synthesis

本仓库收录了围绕 **NISQ（含噪声中等规模量子计算）时代对角线酉矩阵近似综合算法** 这一研究主题的两份文档：一篇 arXiv 论文与一篇本科毕业论文。

## 文件说明

### 1. `2412.01869v1.pdf` — arXiv 论文

**Approximate Quantum Circuit Synthesis for Diagonal Unitary**
Wenqi Zhang, Jinyang Liu, Zixiang Zhou, Shuai Yang（杭州电子科技大学 网络空间安全学院）
arXiv:2412.01869 [quant-ph], Dec 2024

提出一种基于**相位构件（phase gadget）**的近似量子线路综合算法：将对角线酉矩阵的综合问题转化为无向图上的**路径搜索问题**。核心是“相位重要性（phase importance）”的量化与排序——对 Rz 旋转角取绝对值并标准化，再用 logistic 函数放大关键阈值附近的差异，以此衡量每个相位构件对最终误差的贡献；随后结合路径选择、路径扩展（借鉴 Hamiltonian 路径判定思想）与“死胡同”处理策略，构造出尽量优的相位构件排序，从而在给定 CNOT 门预算下合成误差最小的对角线酉矩阵。实验最高可综合 **15 比特**，在 0%–12% 的误差区间内平均可将 CNOT 门节省比例做到误差的 3.2 倍（定义为“算法效用比”），15 比特下运行时间约 561.71 秒。

### 2. `NISQ时期下针对对角线酉矩阵的近似综合算法研究.pdf` — 本科毕业论文

同一研究主题的中文毕业设计论文，采用了另一套核心算法：基于**蒙特卡洛树搜索（MCTS）**——将每个候选相位位置作为搜索树节点（属性包括 value、child、ancestor、father、visit、quality），设计融合“现有解质量 + 未来潜力（按汉明距离加权）+ 探索项”的选择公式，并配合搜索树剪枝、参数（α）梯度下降优化（在树搜索完成后统一进行以节省时间开销）等策略。最高可综合 **12 比特**，例如 10 比特、允许误差 11.96% 时可节省 203 个 CNOT 门。论文附录还介绍了配套的可视化软件包（`start.py`/`search.py`/`TSP.py`/`draw.py` 等模块，基于 `easygui` 做交互界面、`qiskit` 做线路绘制），并将“贪心选取高权重相位 + TSP 变体求解剩余排序”列为未来工作展望。

## 两者关系

两份文档研究同一问题（NISQ 时代对角线酉矩阵的近似综合），共享相同的理论基础：相位构件、Walsh-Hadamard 变换（关联目标相位向量 λ 与 Rz 旋转角向量 α）、误差的二范数定义 D(U_C, U_T)，以及“给定 CNOT 门数求最小误差 / 给定误差求最小门数”两类任务设定。

毕业论文是较早完成的版本，核心算法为 MCTS 启发式树搜索；毕业论文未来工作中提出的“贪心 + TSP 变体”思路，后续演化为 arXiv 论文中更成熟、确定性的“相位重要性 + 图路径搜索”算法——用它取代了 MCTS，将可综合的比特规模从 12 提升到 15，并新增了“算法效用比”等定量评估指标与更完整的实验数据（Table 1–3）。
